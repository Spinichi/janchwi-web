# Backend

잔취 서비스의 백엔드 애플리케이션입니다.  
도메인 중심 설계와 확장 가능한 구조를 목표로 합니다.

## 기술 스택
- Java 25
- Spring Boot
- MySQL
- Redis (예정)
- RabbitMQ (예정)

## 주요 역할
- 사용자 인증 및 권한 관리
- 술, 리뷰, 댓글, 게시글 API 제공
- 실시간 채팅 및 이벤트 처리
- 이미지 / 영상 업로드 연동을 위한 API 제공

## 아키텍처 방향
- REST API 기반 설계
- 도메인 단위 패키지 구조
- 비즈니스 로직과 인프라 계층 분리
- Redis를 활용한 캐싱 및 세션 관리 (검토 중)
- RabbitMQ를 활용한 비동기 이벤트 처리 (검토 중)

## 데이터베이스
- MySQL 기반 RDB 설계
- 리뷰, 댓글, 채팅 메시지 등 관계 중심 모델링

## 실행 방법
```bash
./gradlew bootRun
```
